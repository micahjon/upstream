Stream processing, for humans.

Upstream makes stream processing both approachable and collaborative

Join the beta!

Upstream takes the best pieces of stream processing and puts them in a single place. We provide a simple interface for creating channels of events, filtering and aggregation transformations, live updating charts, and creating external responses such as sending emails or triggering webhooks. All that takes a few minutes, tops.


Create your stream

Just sign up with either an email or a google account, create a channel, and start firing POST requests.

$ curl \
  -H "Content-Type: application/json" \
  -X POST \
  -d '{"cpu":".23","memory":".13","load":"4"}'
  https://api.upstreamapp.io/hook/YOUR_SECRET_KEY
